Case Studies - Selected Work

Tanzanian Well Data

This is a shiny app which shows the location of wells in Tanzania. The app was developed to help with a DrivenData machine learning competition. Users can visualize the locations of wells by functional status and well funder, and display the results by individual well or grouped by location. Additional features will be added as necessary. The intent is to help show patterns in the data which might lead to accurate prediction of well functional status.

The Logistic Map

The logistic map is a nonlinear difference function which determines the population in a given period based on the population of the previous period. The equation is simple, but the results are complex - at certain parameters. the logistic map demonstrates chaotic behavior. This is a simple look at the logistic function, including the R code necessary to generate a bifurcation chart.

Optimization in Julia with R

Both the Julia and R languages are used for operations modeling. Minimizing (or maximizing) an operational function with constraints can be accomplished in both languages, but they both have their own benefits. Julia's ability to call R functions directly allows for the best of both worlds. This is a first look at the code to solve a linear programming problem in Julia, using R code.

Coinflip Simulator

This is a simple shiny application which simulates the flipping of a number of coins. The probability of a heads can be selected by the user, so the coin's 'fairness' is not guaranteed. The app tracks the cumulative number of successes and the cumulative sum of the outcomes, assuming a success is +1 and a failure is -1.

More About Me
